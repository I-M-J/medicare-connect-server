The robust Express and MongoDB-based REST API powering the MediCare Connect healthcare platform. This backend handles everything from secure authentication verification and complex database queries to payment processing.

## âś¨ Key Features & Architecture

- **Secure Authentication Verification**: Implements custom JWT validation using `jose-cjs` to securely decode and verify tokens generated by Better-Auth from the client side.
- **Strict Role-Based Access Control (RBAC)**: Custom middleware layers ensuring that `admin`, `doctor`, and `patient` users are strictly restricted to their authorized endpoints.
- **High-Performance Database Operations**: Utilizes the native MongoDB Node.js driver (avoiding heavy ORMs like Mongoose) for highly optimized, direct database interactions and aggregations.
- **Automated Data Synchronization**: Intelligent webhook-like endpoints that synchronize Google OAuth users from the auth provider directly into the core relational collections.
- **Secure Payment Processing**: Integrated with the Stripe API to securely generate payment intents and process appointment booking transactions.

## đź“¦ Local Setup & Installation

1. Clone the repository and navigate to the server directory.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables by creating a `.env` file:
   ```env
   PORT=5000
   MONGODB_URI=mongodb+srv://<your-cluster-url>
   CLIENT_URL=http://localhost:3000
   STRIPE_SECRET_KEY=sk_test_...
   ADMIN_EMAIL=admin@medicare.com
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
